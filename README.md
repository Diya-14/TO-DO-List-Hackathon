# Smart Todo CLI

A command-line task management tool featuring Natural Language Processing and AI-driven task organization.

## Features
- **Natural Language Input**: Add tasks like "Buy milk tomorrow at 5pm urgent"
- **AI Organization**: Automatically cluster tasks by semantic similarity
- **Offline First**: No internet required, local JSON storage
- **Rich Interface**: Beautiful terminal output tables

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python src/main.py --help
```
